# Problem Statement

## Overview

Build an **AI-powered Review Discovery Engine** for Spotify that collects, analyzes, and visualizes user feedback from multiple public data sources to uncover insights about music discovery and recommendation quality.

## Objective

Understand the experience users have with music discovery on Spotify, specifically:

- Why users struggle to discover new music
- What frustrations they have with Spotify's recommendation system
- How different user segments behave
- What unmet needs consistently emerge from user conversations

## Dashboard Objectives

The AI workflow should process the collected data using LLM/NLP techniques and present an interactive dashboard that answers questions such as:

- Why do users struggle to discover new music?
- What are the biggest frustrations with Spotify recommendations?
- What listening behaviors lead users to repeatedly play the same music?
- Which user segments face different discovery challenges?
- What unmet needs appear consistently across reviews and discussions?
- What themes, sentiments, and feature requests emerge most frequently?
- What actionable product opportunities can Spotify pursue to improve meaningful music discovery?

## Final Deliverable

A clean, insight-driven dashboard containing:

- Visualizations
- AI-generated summaries
- Key themes
- Sentiment analysis
- User segmentation
- Recurring pain points
- Product recommendations

These outputs should help Spotify's **Growth Product team** make data-driven decisions.
